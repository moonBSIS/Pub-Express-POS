## 🚀 FastAPI POS System (Dockerized)

### ✅ Requirements

- Docker
- Docker Compose

### 🔧 Setup

1. Clone the project
2. Run: `docker-compose up --build`
3. Visit:
   - Frontend: http://localhost:5173
   - Backend: http://localhost:8000/docs

### 🐳 Docker Notes

- Backend: FastAPI + SQLite
- Frontend: Vue + Vite + Nginx
